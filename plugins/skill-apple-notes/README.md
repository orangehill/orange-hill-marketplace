# skill-apple-notes

Manage Apple Notes via the `memo` CLI on macOS (create, view, edit, delete, search, move, and export notes). Use when a user asks Claude Code to add a note, list notes, search notes, or manage note folders.

## Installation

```bash
claude skill install skill-apple-notes
```
